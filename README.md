# revenprox.dotcommoners.com

Landing + marketing site for **[RevenProx](https://github.com/dotcommoners/revenprox)** — a high-performance Zig SSE proxy for real-time streaming at scale, by [dotcommoners](https://github.com/dotcommoners).

Built with [Astro](https://astro.build) as a static, SEO/AI-search-optimized site. Served under the docs domain at **/revenprox/**.

## Develop

```bash
npm install
npm run dev
npm run build    # static output to ./dist (base path: /revenprox/)
```
